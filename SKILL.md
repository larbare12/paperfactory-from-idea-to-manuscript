# 论文写作助手 (Paper Assistant)

## 快速开始

本 Skill 旨在实现：**人只需提供 idea，代码、验证、论文全流程自动化**。

### 核心功能

1. **论文结构规划** - 基于 IMRAD 标准生成大纲
2. **写作规范检查** - 语言风格、术语使用、时态一致性
3. **实验设计审查** - 完整性检查清单
4. **格式规范管理** - Overleaf 项目命名、参考文献格式
5. **Prompt 模板库** - 大纲生成、段落润色、摘要撰写

## 使用指南

### Step 1: 初始化项目

创建 Overleaf 项目，命名格式：
```
姓名_标题_类型0x_年月

例：Y.Nie_EM-SDPD_final_2409
    Z.Wang_DRL-AF_paper02_2407
```

### Step 2: 规划论文结构

使用 Prompt 模板生成大纲（参见 [PAPER-WRITING-GUIDE.md](refence/PAPER-WRITING-GUIDE.md) 第 7.1 节）。

### Step 3: 撰写与迭代

- 先写方法/实验（最容易）
- 最后写引言和摘要（需要全文视角）
- 使用段落润色 Prompt 优化语言

### Step 4: 格式检查

提交前自查清单：
- [ ] 数学符号首次出现已定义
- [ ] 缩写首次出现已说明全称
- [ ] 所有 \ref{} 引用正确
- [ ] 参考文献按字母序排列
- [ ] 图表符合三线表规范
